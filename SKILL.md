# claw-lint

Project health checker for AI agents - validates projects are ready to ship

## Usage

```bash
claw-lint [options]
```

## Options

- `--json`
- `--fix`
- `--strict`
- `--porcelain`
- `-j`
- `-f`
- `-s`
- `-h`
- `-v`

## When to Use

Use this tool when you need to project health checker for ai agents - validates projects are ready to ship.

## Notes

- Run `claw-lint --help` for full usage information
